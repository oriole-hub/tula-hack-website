# TEAM BUILDER

Веб-интерфейс для работы с командами, сотрудниками, оценками, кадровым резервом и внутренней HR-аналитикой.

Проект собран на `React + TypeScript + Vite`, использует локальные мок-данные и готов к дальнейшему подключению backend без смены основной структуры интерфейса.

## Что внутри

- авторизация и регистрация компании
- дашборд с ключевыми метриками и матрицей потенциала
- разделы `Команды`, `Сотрудники`, `Оценки`, `Идеальный профиль`, `Кандидаты`
- экраны `Риски`, `Карта ролей`, `Кадровый резерв`, `Преемственность`, `Уведомления`
- ролевая навигация для администратора, HR, руководителя и сотрудника
- единая стилистика интерфейса с тёмной оболочкой и светлыми рабочими панелями

## Технологии

- `React 18`
- `TypeScript`
- `Vite`
- `React Router`
- `React Hook Form`
- `Zod`

## Запуск проекта

Установка зависимостей:

```bash
npm install
```

Запуск в режиме разработки:

```bash
npm run dev -- --host 127.0.0.1
```

После запуска открой:

```text
http://127.0.0.1:5173
```

Сборка production-версии:

```bash
npm run build
```

Предпросмотр production-сборки:

```bash
npm run preview
```

## Данные

Сейчас проект работает на локальных мок-данных.  
Они подключены через внутренний state-слой, поэтому интерфейс уже можно смотреть, демонстрировать и дорабатывать без backend.

При будущем подключении API логика экранов и структура приложения останутся пригодными для расширения.

## Роли в интерфейсе

- `Администратор компании` — полный доступ
- `HR` — работа с людьми, подбором, аналитикой и резервом
- `Руководитель` — команды, сотрудники, оценки и риски
- `Сотрудник` — только доступные персональные сценарии
